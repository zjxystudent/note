## 题解

**排序**

执行用时：_116 ms_ 内存消耗：_46.2 MB_

时间复杂度：_O(n)_ 空间复杂度：_O(1)_
```javascript
/**
 * @param {number[]} nums
 * @return {number}
 */
var arrayPairSum = function (nums) {
    const length = nums.length;
    nums.sort((a, b) => {
        return a - b
    })
    // console.log(nums)
    let result = 0;
    for (let index = 0; index < length; index += 2) {
        result += nums[index]
    }
    return result
};
```
