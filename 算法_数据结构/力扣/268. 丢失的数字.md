## 排序
 
  执行用时：_56ms_ 内存消耗：_15.8MB_
  ```python
  class Solution:
    def missingNumber(self, nums: list[int]) -> int:
        nums.sort()
        size = len(nums)
        while size:
            if nums[size-1] != size:
                return size
            size = size-1
        return size
result = Solution()
print(result.missingNumber(nums=[1]))  # 0
print(result.missingNumber(nums=[9, 6, 4, 2, 3, 5, 7, 0, 1]))  # 
```


## 计数

执行用时：_40ms_ 内存消耗：_16MB_
```python
class Solution:
    def missingNumber(self, nums: list[int]) -> int:
        size = len(nums)+1
        now=[0]*size
        for i in nums:
            now[i]=1
        return now.index(0)
result = Solution()
print(result.missingNumber(nums=[1]))  # 0
print(result.missingNumber(nums=[9, 6, 4, 2, 3, 5, 7, 0, 1]))  # 8
```
